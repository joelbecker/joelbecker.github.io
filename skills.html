Skills Summary

Whether I'm developing new tools or making statistical inferences, I'm all about data.

Data Science

I excel at creating reproducible insights using statistical inference and data visualization in Python and R. Past analytics projects have involved analyzing social networks, topic modelling text, and large-scale multi-phase survey research. This work involved extracting, cleaning, and integrating data from multiple sources.

Software Development

I have two years' experience designing flexible and coder-friendly Python packages for data extraction and data integration. I have made key improvements to the open-source data integration toolkit recordlinkage by adding new features and eliminating a major performance bottleneck. I also created the data extraction package tidyextractors from the ground up.

Research Design and Implementation

I have three years' experience designing and implementing investigations of complex social phenomena. Past research includes how supervisors' race and gender affect workplace outcomes, and changes in the intellectual diversity of scientific research teams.

Technical Skills

My favourite tools are the languages I use to make data talk.

Python

As an experienced Python programmer, I don't just use Python for data analytics — I also develop data extraction and data integration tools which streamline the data science workflow. I am an active contributor to the recordlinkage data integration toolkit, and created the tidyextractors data extraction tool from the ground up.

R

I have three years' experience using R's powerful libraries for data analysis and data visualization. In addition to data management, data visualization with ggplot2, and classical statistical inference, I use R for structural social network analysis, textual topic modelling, and psychometrics.

SQL

I am proficient in the use of SQL to query and aggregate data. I am also knowledgeable of database design, having studied topics in database theory such as data normalization and entity resolution. I frequently use these concepts to design the data flow of data science solutions.

C++

I am proficient in developing small- and medium-sized object-oriented programs in C++. My academic training in C++ included a collaborative software development project, best practices for object oriented software development, and core language features.

MatLab

I am proficient in MatLab, having completed course-based assignments involving data interpolation, error analysis, image compression, signal processing, and computational linear algebra.

Scala

I have basic proficiency in Scala, having completed course-based assignments reviewing the fundamentals of functionally and object-oriented programming in Scala. I am currently designing a high-performance data integration tool, which I hope to implement in Scala.

HTML

Git

Miscellaneous
